## Task 1 - HTTP Methods

### GET
Used to retrieve or read data from the server. It does not modify any data.

**Example:**
GET /projects

---

### POST
Used to create a new resource by sending data to the server.

**Example:**
POST /projects

---

### PUT
Used to completely replace an existing resource with new data.

**Example:**
PUT /projects/1

---

### PATCH
Used to update only specific fields of an existing resource without replacing the whole resource.

**Example:**
PATCH /projects/1

---

### DELETE
Used to remove a resource from the server.

**Example:**
DELETE /projects/1
