
## Task 2 - HTTP Status Codes

200 OK - Request completed successfully - GET Method
201 Created - New resource created successfully - POST Method
400 Bad Request - Invalid or incorrect request from the client 
401 Unauthorized - User is not authenticated 
403 Forbidden - User is authenticated but doesn't have permission 
404 Not Found - Requested resource does not exist - GET Method
500 Internal Server Error - Something went wrong on the server
